get

(
  • name
  • options
)
Variant static

Defined in cookie/js/Cookie.js:234

Returns the cookie value for the given name.

Parameters:

  • name String

    The name of the cookie to retrieve.

  • options Function | Object

    (Optional) An object containing one or more cookie options: raw (true/false) and converter (a function). The converter function is run on the value before returning it. The function is not used if the cookie doesn't exist. The function can be passed instead of the options object for backwards compatibility. When raw is set to true, the cookie value is not URI decoded.

Returns:

Variant: If no converter is specified, returns a string or null if the cookie doesn't exist. If the converter is specified, returns the value returned from the converter or null if the cookie doesn't exist.
